# Load Excavator: Privacy policy

Welcome to the Load Excavator app for Android!

Load Excavator is an app developed by Eric Holzapfel for use by approved customers.

This document is a summary of the privacy policy

I have not programmed this app to collect any personally identifiable information. All data created by the app (ex. load tracking data) is stored locally only and can be deleted by the user (example, by using the 'delete' button on the app main screen.
